# Текстовая одиночная RP-игра с простым ИИ

Это минимальный шаблон однопользовательской текстовой ролевой игры (RPG) с простым встроенным ИИ персонажа.
Репозиторий включает:
- `game.py` — основной игровой цикл и движок
- `ai.py` — простой ИИ (правила + опция для интеграции с OpenAI)
- `requirements.txt` — зависимости (опционально)
- `LICENSE` — лицензия MIT
- `examples/` — папка с примерами сценариев

Как использовать
1. Скачайте и распакуйте репозиторий.
2. (Опционально) создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate   # Linux / macOS
   venv\Scripts\activate    # Windows
   pip install -r requirements.txt
   ```
3. Запустите игру:
   ```bash
   python game.py
   ```

Интеграция с современным ИИ
- В `ai.py` есть заглушка для интеграции с OpenAI (или другими провайдерами). Там показано, как отправлять подсказку и получать ответ — **требуется** собственный API-ключ и установка библиотеки (например, `openai`).
